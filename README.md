A small CLI tool to help find and track potential leads for Trucker Pal.

## What it does (MVP)
- Takes a list of communities (subreddits, forums, FB groups, etc.) and produces a posting plan.
- Lets you paste in thread URLs and generates:
  - suggested comment reply
  - suggested DM follow-up (opt-in only)
  - tags (parking, weather, fuel)

## Run
```bash
npm install
npm start
```

## Notes
This tool is designed to avoid scraping private data or generating spam.
